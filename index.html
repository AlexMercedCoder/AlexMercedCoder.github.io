Django/Mezzanine CMS Blog

This project was about learning how to install django based Mezzanine CMS and deploy it to a fresh Linode VPS. After much trial I learned a lot about the command line and was able to deploy it using NginX and uWSGI. I also installed the Django podcasting app to try installing 3rd party Django apps into Mezzanine. All went well. Like the other Django site I didn't style it till after deployed so did a lot of the visual editing using nano which was a little cumbersome. I also installed the SSL certificate after deployment so I got it up and running head to toe.
